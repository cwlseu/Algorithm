# 树

## 二叉树

### 满二叉树
深度为h的且有2^h - 1个节点的二叉树

              1
            /   \
           2     3
          / \   / \
         4   5 6   7

### 完全二叉树

              1
            /   \
           2     3
          / \   / 
         4   5 6

              1
            /   \
           2     3
          / \    
         4   5 

              1
            /   \
           2     3
          /    
         4 

如果完全二叉树的一个父亲节点编号为k, 那么左儿子节点编号为2k, 右儿子编号为2k+1. 如果已知一个节点编号为x, 那么其父亲节点编号为x/2


## 最小堆
特殊的完全二叉树。完全二叉树一个性质，最后一个非叶子节点是第n/2个节点。
